# TravelGo
Android Studio ile geliştirilmiştir Mobil Uygulamalar Projesidir

🚀 Öne Çıkan Özellikler
Gelişmiş Keşif Ekranı (SearchFragment): Popüler ve öne çıkan dünya destinasyonlarının (Ayasofya, Eyfel Kulesi, Kolezyum, Tokyo vb.) kategorize edilerek listelenmesi.

Akıllı Arama Çubuğu: Kullanıcı dostu arayüze sahip, yerel XML çizimleriyle optimize edilmiş arama ve filtreleme alanı.

Dinamik Bilet Onay Sistemi: Kullanıcının seçtiği destinasyona göre sistem tarafından otomatik olarak atanan benzersiz tarih, saat, konum ve fiyat bilgileriyle bilet oluşturma mekanizması.

Dijital Bilet Cüzdanı (PalmFragment): Satın alınan biletlerin yerel veritabanından anlık olarak okunarak kullanıcıya özel tasarlanmış kartlarla (RecyclerView & Adapter) listelenmesi.

İnteraktif Bilet Detay Ekranı (MainActivityTicket): Cüzdandaki biletlere tıklandığında açılan; seyahat planına ait özel açıklamaları, konum verilerini ve kapı girişi için üretilen dijital QR kodu barındıran bilet görünümü.

🛠️ Kullanılan Teknolojiler ve Mimari
Geliştirme Ortamı: Android Studio

Programlama Dili: Java

Veritabanı: SQLite (SQLiteOpenHelper ile users, mekanlar ve biletler arası ilişkisel veri yönetimi)

Tasarım ve UI/UX: XML (Material Design Bileşenleri, CoordinatorLayout, NestedScrollView, ConstraintLayout)

Mimari Yapı: Single-Activity Mimarisi (Fragment konfigürasyonları ve dinamik FragmentManager geçişleri)
